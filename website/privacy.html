Privacy Policy for Count Rewards Bot

1. Information We Collect

Count Rewards Bot collects only the minimum data necessary to provide its services:

Data Type Purpose Retention
Discord Server IDs Identify which servers use the bot Until bot is removed from server
Channel IDs Track designated counting channels Until configuration is changed
User IDs Track reward providers and last counter Until reward is claimed or data is reset
Count Numbers Track counting progress Indefinitely for historical purposes
Reward Descriptions Store reward information Until reward is claimed or deleted

2. How We Use Your Information

We use the collected information to:

  • Provide counting channel monitoring services
  • Manage and track rewards for counting milestones
  • Send notifications when reward goals are reached
  • Celebrate counting achievements and milestones

3. Data Storage and Security

Your data is stored securely:

  • Data is stored in an encrypted SQLite database
  • Database is hosted on secure cloud infrastructure
  • Access is restricted to authorized bot operations only
  • No personal information beyond Discord IDs is collected

4. Data Sharing

We do not share your data with third parties, except:

  • When required by law or legal process
  • With Discord's API for bot functionality
  • With our hosting provider for technical operations

5. Your Rights

You have the right to:

  • Access: Use the /cr-me command to view all data we store about you
  • Deletion: Use the /cr-remove-me command to permanently delete all your personal data
  • Correction: Request correction of inaccurate data through our support channels
  • Portability: Request your data in a machine-readable format through our support channels
  • Server-level deletion: Remove the bot from your server to delete all server-related data

6. Self-Service Data Management

We provide easy-to-use commands for managing your personal data:

  • /cr-me: View all personal data we have stored about you
  • /cr-remove-me: Permanently delete all your personal data from our database
  • These commands work instantly and don't require server administrator permissions
  • Data removal is permanent and cannot be undone
  • You can continue using the bot normally after data removal

7. Data Retention

We retain your data as follows:

  • Server configuration data: Until bot is removed from server
  • Reward data: Until rewards are claimed, manually deleted, or removed via /cr-remove-me
  • Count history: Indefinitely for service functionality
  • User associations: Until rewards are claimed, data is reset, or removed via /cr-remove-me
  • Personal data can be removed at any time using the /cr-remove-me command

8. Children's Privacy

The Bot is not intended for use by children under 13. We do not knowingly collect personal information from children under 13. If you believe we have collected information from a child under 13, please contact us immediately.

9. International Data Transfers

Your data may be transferred to and processed in countries other than your own. We ensure appropriate safeguards are in place to protect your data during such transfers.

10. Changes to This Policy

We may update this Privacy Policy from time to time. We will notify users of any material changes by posting the new Privacy Policy on this page. Your continued use of the Bot after changes are posted constitutes acceptance of the new policy.

11. Contact Us

If you have any questions about this Privacy Policy or want to exercise your rights, please contact us:

  • Self-Service: Use /cr-me to view your data or /cr-remove-me to delete it
  • Through our GitHub repository
  • Via Discord server (if available)
  • By removing the bot from your server (for server-level data deletion)
